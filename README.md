# Dungeon-
Dungeon of Madness transforms the Python terminal into a cinematic horror experience. Featuring retro 8-bit art, dynamic audio mixing, and voice-synced text, every choice has consequences. Survive a toxic forest, a dark cave, face the Shadows and hills. Outsmart the narrator, solve the final riddle,and claim the ultimate treasure or perish trying.
Dungeon is an immersive Python-based adventure game that blends logic, creativity, and atmosphere. Players explore mysterious environments, guided by symbolic objects and dramatic choices that shape their journey. With integrated audio effects and custom visuals, the game delivers a cinematic experience right inside the terminal.
Built with Python and packaged into a playable .exe, Dungeon is designed to be accessible to everyone — no setup required. Whether you’re curious about interactive fiction, creative coding, or simply want to enjoy a fun challenge, this project showcases how Python libraries can be used to craft engaging, story-driven gameplay.
